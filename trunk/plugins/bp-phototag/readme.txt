Photo Albums for BuddyPress with friend tagging (like facebook). Includes Posts to Wire, Member Comments, and Gallery Privacy Controls.


== Description ==
This plugin adds full photo album capabilities to BuddyPress, mimicking Facebook's photo app!(including Face-Tagging)

This is an early release version of the plugin. You should test it extensively before using it on a production site. It provided on an “as is” basis without any warranty. We are not liable for any damage or losses.


== Installation ==

1. Upload `bp-phototag` to the `/wp-content/plugins/` directory or use automatic installation from wp plugin panel
2. Activate the plugin through the 'Plugins' menu in WordPress

1. **A tagged Photo** - The tag will become visible on a mouseover. 
2. **Tagging a Photo** - Creating a new Photo Tag.
3. **Activity Post** - When a member is tagged in a photo, it will show in their activity stream.
